# Finnish Law Ontology Viewer

Interactive Streamlit app to visualize Finnish law ontologies as knowledge graphs.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Copy your ontology YAML file to this folder (e.g., `work_accident_ontology.yaml`)

3. Run the app:
```bash
streamlit run ontology_viewer.py
```

## Requirements

- streamlit
- pyvis
- networkx
- pyyaml
